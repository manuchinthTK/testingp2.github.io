Minecraft 1.21.5 Server Download

Server Files

Download these files to run your own server:

Download server.jar
Download eula.txt
Download Windows start.bat
Download Linux/macOS start.sh

Installation Instructions

  1. Download all files above
  2. Place them in the same folder
  3. For Windows: Double-click start.bat
  4. For Linux/macOS: Run chmod +x start.sh then ./start.sh
  5. The server will generate files on first run
